JSIV - Jayde's Simple Image/Icon Viewer

Going to learn to use the 

    Graphics2D g2D = g;
    g.zoom();
    
(and related methods) instead of calculating everything by hand myself.
Collaborating with ChatGPT, it showed me a code snipped that simplified my
zoom display, so I want to see if it actually works or if it was hallucinating.

I don't doubt it will work, just gotta learn it!

I am doing this for 2 reasons:

1) I want to be able to do this more simply and reliably than I have been.

2) I want to make an icon viewer to view icons at larger size more easily and
without the blur that standard image viewer causes.  

So this is what this learning application is going to do:

Jayde's Simple Image Viewer (JSIV)

Start with a blank window.  Menu->File->Open to open an image.  RMB to pan,
scroll to zoom, LMB on left half of Viewport to load "previous" image in folder,
LMB on right half of Viewport to load "next" image in folder.  Display image
name in Program Window's title bar.

Also learning Menuing, Action Items and several other things with this.
