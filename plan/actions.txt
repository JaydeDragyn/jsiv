jsiv Actions

These are the Action events that the program will respond to.

Open                    Ctrl-O
Open Next               Ctrl-N
Open Previous           Ctrl-P
Quit                    Ctrl-Q
Pan                     RMB Drag
Move Image              Arrow Keys
Zoom In                 Ctrl-'+'
Zoom Out                Ctrl-'-'
Reset Zoom              Ctrl-0 (zero)
Center Image            Ctrl-C
User Manual             F1
About                   Menu->About

--------------------------------------

Open                    Ctrl-O, Menu->File->Open

Displays the File Dialog for the user to select a file to open.

The Viewport will handle opening files (via a private FileHandler)

---

Open Next               Ctrl-N, Menu->File->Next, LMB in right part of window

Tries to open the next file (in alphabetical order) in the directory from the
current file that is open.  If no "Next file" is available, this option 
should be disabled.

The Viewport will handle opening the next file (via a private FileHandler)

---

Open Previous           Ctrl-P, Menu->File->Previous, LMB in left part of window

Tries to open the previous file (in alphabetical order) in the directory from
the current file that is open.  If no "Previous file" is available, this 
option should be disabled.

The Viewport will handle opening the previous file (via a private FileHandler)

---

Quit                    Ctrl-Q, Menu->File->Quit, [X]

Quits the program.

Handled by the ProgramWindow

---

Pan                     RMB Drag

Anchors the pixel under the mouse pointer and pans the image so that pixel
stays under the mouse pointer until RMB is released.

The Viewport will handle Pan operations.

---

Move Image              Arrow Keys, Ctrl-Arrow Keys

Moves the image in the direction of the arrow key pressed.

If only the Arrow key is pressed, the image moves 1 pixel.

If Ctrl-Arrow is pressed, the image moves 10 pixels.

The Viewport will handle Moving the Image

---

Zoom In                 Ctrl-'+', Menu->View->Zoom In, Scroll forward

Zooms into the image so the pixels appear to become larger.  

If initiated with keyboard or Menu, the zoom will focus on the center of
the Viewport.

If initiated with the scroll wheel, will focus on the pixel under the
mouse pointer.

The Viewport will handle Zooming In

---

Zoom Out                Ctrl-'-', Menu->View->Zoom Out, Scroll backward

Zooms out of the image so the pixels appear to become smaller.

If initiated with keyboard or Menu, the zoom will focus on the center of
the Viewport.

If initiated with the scroll wheel, will focus on the pixel under the
mouse pointer.

The Viewport will handle Zooming Out

---

Reset Zoom              Ctrl-0, Menu->View->Reset Zoom

Resets the Zoom level to the maximum zoom needed to display the image as
large in the Viewport as possible without any pixels disappearing outside
the window.  Also Centers the image (see below).

The Viewport will handle Resetting the Zoom

---

Center Image            Ctrl-C, Menu->View->Center Image

Pans the image so the center pixel of the image (or the left/top-center
pixel) lines up with the center pixel of the Viewport (or left/top-center
pixel of the Viewport).

The Viewport will handle Centering the image

---

User Manual             F1

Displays a non-modal window with a wiki-style user manual to explain how 
this program works and provide quick reference to the commands.

The ProgramWindow will handle displaying the User Manual via a UserManual object

---

About                   Menu->Help->About

Displays a modal dialog with information about the program author, dev
environment, etc.

The ProgramWindow will handle displaying the About dialog.
