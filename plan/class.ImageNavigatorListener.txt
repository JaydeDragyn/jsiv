jsiv ImageNavigatorListener

The ImageNavigatorListener is an interface that another class uses to respond
to the ImageNavigator's results when it is asked to do something.

The ImageNavigator will emit the following information:

    void navigationAvailabilityChanged(boolean available);
    void newImageLoaded(String imageName,
                        BufferedImage image,
                        int index, int total);

-------------------------------------------------------------------------------

navigationAvailabilityChanged(boolean available)

This will advise the listener whether navigation is available
(available == true) or not (available == false).  This is a way to tell other
components that the ImageNavigator will or will not honor a request to
openNext() or openPrevious().  (openNext() and openPrevious() will attempt
to open a file or ignore the request as appropriate whether other components
respect this message).

-------------------------------------------------------------------------------

newImageLoaded(String imageName, BufferedImage image, int index, int total)

This will advise that a new image has been loaded, provide the image name,
the actual image data as a BufferedImage, and the index and total number of
image files in the same folder.
