jsiv JSIVSplash

The JSIVSplash class is responsible for creating the splash image.

This is a static class that provides only one public method:
    public static BufferedImage getSplashImage();

When called, it will check to see if the splash image has been created yet,
if not it will create it (lazy load), and then will return the image.

The splash image is created using Graphics2D methods on a BufferedImage
so that the program does not need to rely on an external file/resource
for the splash image.
