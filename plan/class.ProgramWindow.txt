jsiv ProgramWindow

This class acts as the Controller of the program, connecting all of the other
classes together and providing a JFrame to attach them to.

As the Controller, it has knowledge of all of the other classes, but does not
provide any public facilities for other classes to call (except for the
constructor which the JSIV class calls when it creates a ProgramWindow).

Instead, the ProgramWindow implements the ViewportListener and
ImageNavigatorListener interfaces so it can respond to those classes
requesting actions or providing information.

This ProgramWindow also sets up the program menu and the InputMap and
ActionMap to handle all of the non-mouse-gesture input.
